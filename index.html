To-Do Application

This project is a full-stack "to-do" application developed with Spring Boot (Java) for the backend and React.js with MUI for the frontend. Key features include:
Backend: Manages routes, data validation, business logic, and data persistence with JPA. Configured CORS for seamless frontend API access.
Frontend: Built with React.js and styled using MUI, providing a responsive and user-friendly interface.
Deployment: Automated using GitHub Actions to build Docker images and deploy them on AWS EC2. Configured Nginx to serve the frontend and proxy API requests to the backend, which connects to an AWS RDS MySQL database.
Functionality: Includes adding, editing, deleting to-dos, assigning items to categories, and filtering based on category. Overcame challenges by writing custom SQL queries in JPA to fetch to-dos by category.
Explore this project to see how Spring Boot and React.js come together to deliver a robust to-do application.

Luxe Threads - E-Commerce Platform for Men's Clothing

This React.js-based E-Commerce platform offers a seamless shopping experience for men's clothing. Key features include:
Home Page: Displays a grid of products.
Product Details: View details, select sizes, and add items to the cart with notification feedback.
Cart Page: Adjust quantities, view the total order value, and access a pending "Checkout" button.
Data Management: Uses Firebase Realtime Database, with data seeded from the H&M API via a custom script.
Explore how React.js and Firebase create an engaging shopping experience.

Google Books Clone

Navigate Pages: Use pagination to browse through multiple result pages.
View Book Details: Click "Quick Preview" to see more details about each book in a modal.
Explore this project to see a functional clone of Google Books with enhanced search and preview capabilities.
